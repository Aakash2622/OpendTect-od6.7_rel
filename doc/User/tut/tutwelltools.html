Tut Well Tools

Intro Smooth

Intro

Here, you can perform some simple operations on well data. For now, only log smoothing is available. First load a well by right-clicking on Well in the OpendTect menu tree, then right-click on an individual well item and select Tut Well Tools.

Log Smoothing

This performs a vertical smoothing of a well log by taking the median of all values in the sample window specified by Sample Gate. Here you cannot overwrite a log. You have to specify a new Log name each time you run the process. If you want, you can delete or rename a log in the Well Manager.
